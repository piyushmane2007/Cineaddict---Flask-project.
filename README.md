# CineAddict 🎬

CineAddict is a movie review web application built using Flask and PostgreSQL.

Users can select a movie, submit a rating, write a review, and receive an email notification after successfully submitting their feedback.

## Features

* Submit movie reviews
* Rate movies from 1 to 10
* Store reviews in a PostgreSQL database
* Email notifications using Mailtrap
* Form validation
* Clean and simple user interface

## Tech Stack

* Python
* Flask
* PostgreSQL
* SQLAlchemy
* HTML
* CSS
* Mailtrap
